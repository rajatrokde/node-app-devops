# 🚀 Minimal DevOps Pipeline Project

This project represents a complete DevOps pipeline, including CI/CD, Dockerization, and IaC, and also provides an option to add monitoring to the pipeline.

---

## 📌 Features

- CI/CD using GitHub Actions
- Dockerization
- IaC using Terraform
- Monitoring using Prometheus and Grafana

---

## 🛠️ Tech Stack Used

- Node.js
- Docker
- GitHub Actions
- Terraform
- AWS EC2
- Prometheus and Grafana
